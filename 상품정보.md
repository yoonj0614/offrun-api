## 개요

상품(Goods) 관련 정보를 조회하는 API 명세서입니다.
상품 ID, 상품명, 페이지 정보를 기반으로 상품 리스트 및 상세 정보를 조회할 수 있습니다.

## API 목록

### 1. 상품 정보 조회 (getGoodsInfo)

상품 ID 또는 상품명 기반으로 상품 목록 및 상세 정보를 조회합니다.

#### 요청

| 항목 | 내용 |
|------|------|
| Method | `GET` |
| URL | `/goods/getGoodsInfo` |
| 권한 | - |

#### Query Parameters

| 이름 | 타입 | 필수 | 기본값 | 설명 |
|------|------|------|--------|------|
| `gid` | `integer` | ❌ | - | 상품 고유 ID |
| `goodsNm` | `string` | ❌ | `""` | 상품명 (부분 검색) |
| `pageSize` | `integer` | ❌ | `20` | 페이지 당 상품 수 |
| `pageNum` | `integer` | ❌ | `0` | 페이지 번호 (0부터 시작) |

#### 요청 예시 (cURL)

```bash
curl -X GET "http://localhost:8082/goods/getGoodsInfo?gid=1&goodsNm=&pageSize=20&pageNum=0"
```

#### 요청 URL 예시

```
http://localhost:8082/goods/getGoodsInfo?gid=1&goodsNm=&pageSize=20&pageNum=0
```

#### 응답 필드 상세

##### `data.goodsInfo`

| 필드 | 타입 | 설명 |
|------|------|------|
| `key` | `string` | 응답 데이터 키 (고정값: `goodsInfo`) |
| `data` | `object` | 상품 데이터 래퍼 |

##### `data.goodsInfo.data`

| 필드 | 타입 | 설명 |
|------|------|------|
| `goodsList` | `array<object>` | 상품 객체 배열 (아래 참조) |
| `totalCount` | `integer` | 조회된 전체 상품 개수 |

## 에러 코드

| HTTP Status | responseMessage | 설명 |
|-------------|-----------------|------|
| 200 | `성공` | 요청 정상 처리 |
| 204 | `상품 정보 없음` | 조회된 상품 데이터가 존재하지 않음 |
| 400 | `잘못된 요청` | 파라미터 누락 또는 타입 오류 |
| 500 | `서버 에러` | 내부 서버 오류 |

## 참고 사항

- `goodsDetail` 필드는 Markdown 형식의 이미지 링크(`![alt](url)`)를 포함하는 긴 문자열입니다. 렌더링 시 Markdown 파서 또는 이미지 URL 파싱이 필요합니다.
- `goodsStock` 필드는 숫자가 아닌 **문자열(string)** 로 반환되므로 클라이언트에서 변환이 필요합니다.
- `pageNum`은 0부터 시작합니다. (0-based index)
- `gid`와 `goodsNm`은 둘 다 선택 파라미터이며, `goodsNm`이 빈 문자열이면 상품명 필터링은 적용되지 않습니다.
